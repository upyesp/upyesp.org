Adding Diagrams To Markdown Using Mermaid

Sunday, August 1, 2021
1 min read
Windows , Linux
Markdown

Mermaid provides a markup language to describe diagrams. Describing diagrams in text brings many advantages.

In the same way that LaTeX and KaTeX provides a text based language to enable elegant math expressions to be drawn, Mermaid provides a language to describe the layout of diagrams.

graph TD
  A --- B
  B-->C[fa:fa-ban forbidden]
  B-->D(fa:fa-spinner);
